## How to run locally : 
step 1 : clone the repo
```bash
git clone https://github.com/sharmapranay38/UMBRELLA-ROOM-VENTURES-INDIA.git
```
step 2 : go to the folder in terminal
```bash
cd UMBRELLA-ROOM-VENTURES-INDIA
```
step 3 : install dependencies
```bash
npm install
```
step 4 : run locally
```bash
npm run dev
```
